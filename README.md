# EasyCut ✂️

## 📖 Sobre o projeto
O EasyCut é uma plataforma web desenvolvida como projeto integrador do curso de Engenharia de Software.  
O objetivo do sistema é facilitar a comunicação entre barbearias e seus clientes, auxiliando na organização de atendimentos e no gerenciamento básico do negócio.

## 🎯 Objetivo
Oferecer uma solução simples e prática para barbearias, permitindo mais controle sobre os atendimentos e melhor experiência para os clientes.

## 🛠️ Tecnologias utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Python

## 🚀 Funcionalidades
- Interface web simples e intuitiva  
- Organização dos atendimentos  
- Comunicação básica entre estabelecimento e clientes  

## 📚 Contexto acadêmico
Projeto desenvolvido como parte do Projeto Integrador do curso de Engenharia de Software, com foco na aplicação prática dos conhecimentos adquiridos durante a graduação.
