📧 Configuração SMTP Gmail - GRATUITO

⚠️ Requisitos:

• Conta Gmail ativa

• Verificação em duas etapas ativada

• Node.js instalado (para backend)

Passo 1: Ativar Verificação em Duas Etapas

1. Acesse: https://myaccount.google.com/security

2. Em "Como fazer login no Google", clique em "Verificação em duas etapas"

3. Siga as instruções para ativar

⚠️ OBRIGATÓRIO: Sem isso, não conseguirá gerar senha de app

Passo 2: Gerar Senha de App

1. Ainda em "Segurança", procure por "Senhas de app"

2. Clique em "Senhas de app"

3. Escolha "Email" como aplicativo

4. Escolha "Outro" como dispositivo e digite "EasyCut"

5. Clique em "Gerar"

6. COPIE A SENHA DE 16 CARACTERES (ex: abcd efgh ijkl mnop)

Passo 3: Instalar Node.js

1. Baixe Node.js: https://nodejs.org/

2. Instale normalmente

3. Abra o terminal/prompt e teste:

node --version

Passo 4: Criar Backend Simples

1. Crie uma pasta para o backend

2. Instale o nodemailer:

npm install nodemailer express cors

3. Crie o arquivo server.js:

const express = require('express'); const nodemailer = require('nodemailer'); const cors = require('cors'); const app = express(); app.use(cors()); app.use(express.json()); // Configuração SMTP Gmail const transporter = nodemailer.createTransporter({ service: 'gmail', auth: { user: 'SEU_EMAIL@gmail.com', // Substitua aqui pass: 'SUA_SENHA_DE_APP' // Substitua aqui } }); // Rota para enviar email de confirmação app.post('/send-confirmation', async (req, res) => { try { const { email, name, confirmationLink } = req.body; const mailOptions = { from: 'EasyCut ', to: email, subject: 'Confirme seu email - EasyCut', html: `

Olá, ${name}!

Obrigado por se cadastrar no EasyCut!

Clique no link abaixo para confirmar seu email:

Confirmar Email

Se o botão não funcionar, copie e cole o link:

${confirmationLink}

` }; await transporter.sendMail(mailOptions); res.json({ success: true, message: 'Email enviado!' }); } catch (error) { console.error('Erro:', error); res.json({ success: false, message: 'Erro ao enviar email' }); } }); // Rota para recuperação de senha app.post('/send-password-reset', async (req, res) => { try { const { email, name, resetLink } = req.body; const mailOptions = { from: 'EasyCut ', to: email, subject: 'Recuperação de senha - EasyCut', html: `

Olá, ${name}!

Você solicitou a recuperação de senha.

Clique no link abaixo para redefinir sua senha:

Redefinir Senha

Se o botão não funcionar, copie e cole o link:

${resetLink}

` }; await transporter.sendMail(mailOptions); res.json({ success: true, message: 'Email enviado!' }); } catch (error) { console.error('Erro:', error); res.json({ success: false, message: 'Erro ao enviar email' }); } }); app.listen(3000, () => { console.log('🚀 Servidor rodando na porta 3000'); console.log('📧 Pronto para enviar emails!'); });

Passo 5: Executar Backend

1. No terminal, na pasta do backend:

node server.js

2. Você verá: "Servidor rodando na porta 3000"
