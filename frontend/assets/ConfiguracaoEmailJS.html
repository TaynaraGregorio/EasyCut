📧 Configuração EmailJS - Envio GRATUITO

Passo 1: Criar Conta EmailJS

1. Acesse: https://www.emailjs.com/

2. Clique em "Sign Up" e crie sua conta gratuita

3. Confirme seu email

Passo 2: Conectar Gmail

1. No dashboard do EmailJS, vá em "Email Services"

2. Clique em "Add New Service"

3. Escolha "Gmail"

4. Faça login com sua conta Gmail

5. Anote o Service ID gerado

Passo 3: Criar Templates

1. Vá em "Email Templates"

2. Clique em "Create New Template"

3. Use este template para confirmação:

Template ID: template_confirmation
Subject: Confirme seu email - EasyCut

Content:
Olá {{to_name}},

Obrigado por se cadastrar no EasyCut!

Clique no link abaixo para confirmar seu email:
Confirmar Email

Se o link não funcionar, copie e cole no navegador:
{{confirmation_link}}

Este link expira em 24 horas.

Equipe EasyCut

4. Crie outro template para recuperação de senha:

Template ID: template_password_reset
Subject: Recuperação de senha - EasyCut

Content:
Olá {{to_name}},

Você solicitou a recuperação de senha.

Clique no link abaixo para redefinir sua senha:
Redefinir Senha

Se o link não funcionar, copie e cole no navegador:
{{reset_link}}

Este link expira em 1 hora.

Equipe EasyCut

Passo 4: Obter User ID

1. Vá em "Account" > "General"

2. Copie seu Public Key (User ID)

Passo 5: Atualizar Código

Substitua os valores no arquivo email-test-service.js:

// Substitua estes valores pelos seus: this.emailjsConfig = { serviceId: 'service_xxxxxxx', // Seu Service ID templateId: 'template_confirmation', // Template de confirmação userId: 'user_xxxxxxx' // Seu User ID };

⚠️ Importante:

• EmailJS é 100% gratuito até 200 emails/mês

• Não precisa de backend

• Funciona direto do frontend

• Emails chegam na caixa de entrada normalmente

✅ Após configurar:

1. Substitua email-service.js por email-test-service.js

2. Atualize as credenciais

3. Teste o cadastro - o email chegará de verdade!
